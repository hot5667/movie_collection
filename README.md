## 프로젝트 소개

**“너이영”** 너는 이미 영화를 보고 있다!  
인기 영화 목록, 팀원별 추천 영화, 영화 검색을 볼 수 있는 자바스크립트를 이용한 플랫폼입니다.

## 구현 기능

- TMDB API 영화 정보 불러오기
- 반응형 웹
- 영화 리뷰 작성, 수정 및 삭제 (Firebase 사용, Firestore)
- 영화 이름으로 검색, 이벤트 처리
- 캐러셀
- 반응형 웹, 모바일, 데스크톱
